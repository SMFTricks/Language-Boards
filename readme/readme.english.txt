[size=12pt][color=navy]Introduction[/color][/size]
[size=8pt]Language Boards mod allows you select a language for each of your boards, from the available list of the forum languages installed and deny board access based on user language.[/size]

[hr]

[size=12pt][color=teal]Features[/color][/size]
[list]
[li][size=9pt]Choose board language or set for any[/size][/li]
[li][size=9pt]Deny access to boards and topics everywhere in the forum based on language[/size][/li]
[/list]
