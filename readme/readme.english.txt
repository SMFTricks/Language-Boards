[size=4][color=#bf307a]Introduction[/color][/size]
[size=1em]Language Boards mod allows you select a language for each of your boards, from the available list of the forum languages installed and deny board access based on user language.[/size]

[size=3][color=#1f4882]Features[/color][/size]
[list]
[li][size=0.9em]Choose board language or set for any[/size][/li]
[li][size=0.9em]Deny access to boards and topics everywhere in the forum based on language[/size][/li]
[/list]
